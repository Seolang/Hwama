## 프로젝트 개발 목적
- 스프링부트를 사용한 유니티 웹 서버 개발 경험
- 포톤 서버를 사용한 멀티플레이 기능 개발
- 웹 서버, 데이터베이스 클라우드 배포 구현

## 담당 업무
- 스프링부트 서버 구축 총괄
- 데이터베이스 설계
- Docker Compose 작성 및 클라우드 배포
- 유니티 HTTP API 통신 기능 개발
- 인 게임 UI 제작
- 포톤 멀티플레이 개발 보조
